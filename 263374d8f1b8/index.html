Windows配置SSH

2022, 6月 01

本文介绍了Windows配置SSH服务器的方法

OpenSSH服务器

特点：开源的钦定软件更稳定，兼容性强，but需要本地账号有密码，空密码反正我没成功过，配置文件很迷，反正我的配置文件没生效过

1.设置-应用-可选功能-添加功能-OpenSSH服务器

2.任务管理器-服务-sshd-启动

3.管理员cmd输入net start sshd

输入你本地账号密码登录就行了

FreeSSHD(软件老，bug多)

1.下载安装

2.配置：User添加个账号，输密码，

Bug1：使用密码登录的时候在authentication中将Password那项选为required；将public key选为disable

Bug2：不支持X11，xshell字母乱蹦

Bug3：putty不支持GBK，所以只能手动调UTF-8，要在注册表里面建立一个键，然而putty的UTF-8还是不支持中文

Bitvise SSH Server

官网

基于Crypto++ 5.3 开发，功能强大，很牛逼，简约漂亮，最新版为8.39，更新日期16 January 2020，网上有注册机可以用，和xshell+xftp配合非常完美，让人感叹商业软件做的就是牛逼，FreeSSHD就是一坨屎

注意：建立虚拟用户的shell命令选Command Prompt，sftp目录选所有，不要作死把文件放在你自己的个人文件夹里面运行，会报权限不够的错误

可以SSH的程序有：易语言程序、ffmpeg、Python
